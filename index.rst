#########################
Welcome to Gutensite Docs
#########################

`Gutensite Docs`_ hosts documentation for the Gutensite API, which includes PHP and Javascript.
We support Sphinx_ documents written with reStructuredText_.
The code is pulled from a Git_ repository, and build the documentation with Sphinx_ and host each Git_ branch on `Gutensite Docs`_.
The page you are currently viewing is the *Universal Directory* for every branch.

All documents used on this website are open source, and `available on github`_.
